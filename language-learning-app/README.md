# Immersive Language Learning System 🚀

## 革新性学习体验

> "Remove everything between the learner and learning."

这不是普通的单词学习应用。这是一个**重新定义了学习体验**的系统。

### 核心创新 ⭐

1. **自动显示机制** - 思考2秒后自动显示单词（或按Space跳过）
2. **完整键盘控制** - 手不离键盘学习（Space, 1/2/3, C）
3. **零摩擦流程** - 看定义→自动显示→按键评分→继续
4. **科学调度** - SuperMemo SM-2 算法
5. **文化洞察** - 跨文化理解支持

## 快速开始

```bash
# 打开主应用
open standalone-app.html

# 或查看独立演示
open immersive-learning.html
```

### 学习流程

```
1. 看到定义："To leave someone or something behind"
2. 开始思考这个词...
3. [2秒后自动显示，或按Space跳过]
4. 看到单词：abandon
5. 按 1/2/3 评分
6. 自动进入下一个
```

**极速模式**: 看定义 → Space → 数字键 → 循环

## 键盘快捷键 ⌨️

| 键 | 功能 |
|---|---|
| `Space` | 跳过等待 / 立即显示 |
| `1` 或 `G` | "I Know It" (Good) |
| `2` 或 `S` | "Somewhat" (Normal) |
| `3` 或 `B` | "New to Me" (Bad) |
| `C` | 切换中文显示 |

## 设计哲学

### "消失的界面"

灵感来自Apple:
- 移除不必要的点击
- 自动化可自动化的
- 保留必要的控制权
- 让用户专注于学习，不是操作

### 学习科学基础

1. **Active Recall** - 定义→思考→验证循环
2. **Spaced Repetition** - 智能复习调度
3. **Flow State** - 消除摩擦，保持专注
4. **Metacognition** - 自我评估与反思

## 技术栈

- **纯JavaScript** - 零依赖，零框架
- **HTML5 + CSS3** - 现代标准
- **localStorage** - 数据持久化
- **SuperMemo SM-2** - 科学算法
- **Golden Ratio** - 美学设计
