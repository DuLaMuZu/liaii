## ✨ 功能特性

### 1. 📊 统计概览
页面顶部显示：
- **总词汇量**：当前演示包含的所有词汇数量（30个）
- **A1级别**：基础入门级词汇数量（20个）
- **A2级别**：初级进阶词汇数量（10个）
- **平均难度**：所有词汇的综合难度平均值

### 2. 🔍 搜索功能
- **支持中文搜索**：输入"桌子"、"快乐"等中文词汇
- **支持英文搜索**：输入"table"、"happy"等英文单词
- **支持定义搜索**：输入英文释义中的关键词
- **实时筛选**：输入即时显示匹配结果

### 3. 🏷️ 筛选功能

#### 按级别筛选
- **全部**：显示所有词汇
- **A1级别**：仅显示基础入门词汇（最简单）
- **A2级别**：仅显示初级进阶词汇（稍难）

#### 按词性筛选
- **名词**：table、book、friend 等
- **动词**：eat、drink、walk 等
- **形容词**：happy、good、interesting 等
- **副词**：always 等
- **代词**：I、you 等

### 4. 📝 词汇卡片详情

每个词汇卡片包含：

#### 基本信息
- **英文单词**：大字号显示，易于阅读
- **词性标签**：n.（名词）、v.（动词）、adj.（形容词）等
- **级别徽章**：A1（绿色）或 A2（橙色）
- **难度指示**：简单/中等/困难（带颜色区分）

#### 释义与例句
- **中文翻译**：多个常用翻译（如：快乐的、高兴的）
- **英文释义**：简单易懂的英文解释
- **例句**：实际使用场景的句子示例

#### 三维评分系统
1. **含义距离**（权重40%）
   - 衡量英中词汇的语义对应程度
   - 分数越低表示含义越接近
   - 范围：0.0-1.0

2. **视觉距离**（权重30%）
   - 衡量英文字母与中文笔画的视觉差异
   - 考虑拼写长度和书写复杂度
   - 范围：0.0-1.0

3. **读音距离**（权重30%）
   - 衡量英文发音的难度
   - 考虑音节数量和发音复杂度
   - 范围：0.0-1.0

4. **综合难度**
   - 根据三个维度加权计算
   - 公式：0.4×含义 + 0.3×视觉 + 0.3×读音
   - 自动判定：简单(<0.35)、中等(0.35-0.55)、困难(>0.55)

### 5. 📄 分页浏览
- 每页显示 **5个** 词汇卡片
- 支持**上一页**、**下一页**按钮
- 显示**当前页码**和总页数
- 智能省略中间页码（如：1 ... 5 6 7 ... 10）

## 📊 评分系统详解

### 含义距离评分标准
- **0.0-0.2**：一对一精确对应（如：table → 桌子）
- **0.2-0.4**：基本对应，略有差异
- **0.4-0.6**：部分对应，存在文化差异
- **0.6-0.8**：显著差异，需要文化理解
- **0.8-1.0**：模糊对应，多义复杂

### 视觉距离计算公式
```
视觉距离 = 1 - exp(-|中文笔画数 - 英文字母数| / 10)
```
- 字母数与笔画数相近时分数较低
- 差异越大分数越高

### 读音距离评估
- **短单词（≤4字母）**：0.70-0.80（如：eat、big）
- **中等单词（5-7字母）**：0.80-0.90（如：happy、table）
- **长单词（≥8字母）**：0.90-1.00（如：understand、interesting）

## 💡 学习建议

### 初学者（A1级别）
1. 先筛选"A1级别"词汇
2. 查找"简单"难度标记的单词
3. 重点关注含义距离低的词汇（易理解）
4. 每天学习5-10个词汇

### 进阶学习者（A2级别）
1. 筛选"A2级别"词汇
2. 可以尝试"中等"难度的单词
3. 注意视觉和读音距离较高的词汇（需要更多练习）
4. 每天学习10-15个词汇

### 词性专项学习
1. 先学习名词（最直观）
2. 再学习动词（常用频率高）
3. 然后学习形容词（增加表达能力）
4. 最后学习副词和代词（语法功能词）

## ❓ 常见问题

### Q1: 为什么有些词汇显示多个中文翻译？
**A:** 英文单词可能有多个常用的中文对应词，系统列出最常用的2-3个翻译，帮助理解词义范围。

### Q2: 综合难度是如何计算的？
**A:** 综合难度 = 0.4×含义距离 + 0.3×视觉距离 + 0.3×读音距离。含义占比最高，因为理解词义是学习的核心。

### Q3: 可以添加更多词汇吗？
**A:** 当前演示版包含30个精选词汇。完整版应用包含250+词汇，需要运行完整的React应用。

### Q4: 如何保存学习进度？
**A:** 演示版不保存进度。完整版应用使用IndexedDB本地存储，可以保存学习进度和错误词汇池。

### Q5: 筛选按钮可以组合使用吗？
**A:** 当前版本支持"级别筛选"或"词性筛选"，两种筛选方式会互相覆盖。完整版支持组合筛选。

## 🎓 系统原理说明

### 三维评分的意义

#### 为什么需要三维评分？
传统的词汇学习只关注词义记忆，忽略了：
1. 文化背景差异（含义距离）
2. 书写难度差异（视觉距离）
3. 发音难度差异（读音距离）

#### 三维评分的优势
1. **个性化学习路径**：根据个人薄弱维度调整
2. **科学评估难度**：综合考虑多个因素
3. **文化理解加深**：关注语言背后的文化差异
4. **记忆效果提升**：多维度记忆更加牢固
